# 15-Minute Demonstration Checklist

## Pre-Demo

- Verify frontend and backend are running.
- Prepare demo account.
- Confirm internet connection and target URL.

## Suggested Flow (15 minutes)

1. Introduction and landing page.
2. Authentication (register/login).
3. Dashboard overview.
4. Session setup and active focus screen.
5. Matchmaking/realtime behavior.
6. Analytics and gamification.
7. Assistant and admin overview.

## Evidence to Show

- Real-time interactions.
- Session tracking and history.
- Role-based access control.
- Deployment readiness and stability.
